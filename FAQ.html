What is this website?

This is a website for creating speedrun guides that can be shared with others.
There is no standard format for speedrun guides at the moment, and It's my goal to change that.

What is speedrunning?

A speedrun is a play-through of a video game performed with the intention of completing it as fast as possible within one's own ability.
Speedruns may cover a whole game or a selected part, such as a single level.

How does this guide work?

This guide breaks a video game down into sections.
Each section has a set of instructions that must be completed until you move unto the next section.

Naming the guide

In the top left of the "Create a guide" page, you can input the name of the guide. Be careful, you can only do this once.
The guide cannot be finished until a name for the guide is entered.

Naming a section

Above the "Add column", "Delete column", and "Add section" buttons is an input field for the name of the section.
You cannot add another section until the current section has been named.

Working with the Instruction Field

Click the "Add column" button to add a column to the Instruction Field. You can have a total of three columns.
Click the "Delete column" button to delete a column. This deletes them in the order that they were added.
Click the "Add section" button when you are finished with the current section and want to add another.
Adding a section deletes any unused textareas, so don't worry about filling up every input field.

Finshing the guide

Once you are finished making the guide, click the "FINISH" button in the top right corner.
Leave any fields that you do not want blank, they will be deleted upon finishing.

Using the guide

When you are ready to use the guide, click the start button. This starts a timer that will keep track of your progress.
Press the "a" button your keyboard to move through the guide easily.
When you do this, it appends the time that you finished the current section below the section name and moves to the next section.

How do I share this with others?

After you have finished making a guide and it is in it's functional state, right click on the page and click "Save as..."
This will give you an HTML file and a folder for the javascript and CSS.
With these, simply share them with your friends or post them on the internet for everyone to use!
